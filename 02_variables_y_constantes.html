Variables y constantes

Variables

Para definir una variable solo hay que poner símbolo de $ delante del nombre que se quiera, símbolo = y asignar un valor
$nombre = "Marianico"; $edad = 32;

Constante

A diferencia de las variables que estas pueden cambiar dentro del programa, las constantes no. Se usan para valores fijos que no van a cambiar nunca. Se definen como define(nombreConstante, valor) y se le llama con el nombre que se le haya puesto echo nombreConstante;

        
            define('pi', 3.141612);
            define('url', 'http://eagj.net');
            echo url; //con esto llamamos a la constante llamada url
