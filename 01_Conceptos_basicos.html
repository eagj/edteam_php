            //Comentario de una sola linea
            #Comentario de una sola linea
            /*
                Comentario multilinea
                para añadir comentarios a
                un trozo
            */

Como insertar código PHP

Todo código php siempre se encierra entre las etiquetas <?php ?>

Dependiendo del contexto este código puede estar elemento por elemento o que ocupe todo un archivo, por ejemplo con funciones

        
            <p> <?php esto es un ejemplo de php en linea; ?> </p>

            <?php 
                //Esto seria un ejemplo de un archivo o un trozo escrito en php, para variables, etc...
                linea de código;
                linea 2 de código;
                linea 3 de código;
            ?>
